# Mock E-Commerce Cart - Vibe Commerce Assignment

## Overview
A full-stack mock e-commerce cart app built for the Vibe Commerce internship assignment.  
Users can browse products, add/remove items from the cart, and perform a mock checkout.

---

## Tech Stack
**Frontend:** React, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**API:** Fake Store API integration  

---

## Features
- View mock products grid  
- Add / remove items from cart  
- Quantity update with total price calculation  
- Checkout form with receipt modal  
- Persistent cart data (MongoDB)
- Clean responsive UI
